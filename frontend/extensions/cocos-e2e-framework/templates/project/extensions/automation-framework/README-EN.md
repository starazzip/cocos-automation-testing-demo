# Cocos Automation Framework

This scaffolded extension provides `db://automation-framework/runtime/test-framework.mjs` for Cocos automation E2E tests.

## Runtime Dependencies

The scaffolded `dist/` runtime uses only Node.js built-in APIs. Cocos Creator can load this extension without running `npm install` inside this directory.

Install local development dependencies only if you need to rebuild or modify the automation-framework extension itself. Do not commit generated `node_modules/`.
